# Kindle SDR Cleaner

Clean useless `.sdr` folders in your Kindle.

## Installation

```bash
pipx install kindle-sdr-cleaner
```

## Usage

```bash
kindle-sdr-cleaner # clean in current folder
kindle-sdr-cleaner /Volumes/Kindle # clean in /Volumes/Kindle
```
